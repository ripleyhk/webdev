Blog Post - October 30th, 2020

This week, I followed an article on freeCodeCamp, written by Zell Liew, where I learned how to build an HTML calculator app using JavaScript.

0

The calculator itself is composed of two primary pieces: the display and the keys. Each is a class stored inside the calculator class. The display is responsible for displaying the current value entered by the user and the calculation. The keys contains all of the buttons for the calculator.

All together, the HTML looks like this:

<div class="calculator">
   <div class="calculator__display">0</div>
   <div class="calculator__keys">
      <button class="key--operator" data-action="add">&plus;</button>
      <button class="key--operator" data-action="subtract">&minus;</button>
      <button class="key--operator" data-action="multiply">&times;</button>
      <button class="key--operator" data-action="divide">&divide;</button>
      <button>7</button>
      <button>8</button>
      <button>9</button>
      <button>4</button>
      <button>5</button>
      <button>6</button>
      <button>1</button>
      <button>2</button>
      <button>3</button>
      <button>0</button>
      <button data-action="decimal">.</button>
      <button data-action="clear">AC</button>
      <button class="key--equal" data-action="calculate">=</button>
   </div>
</div>

CSS is used to center, color, and position elements. Of course, the keys are displayed using a grid layout.

The heavy lifting for this project is in the scripting. All functionality is triggered by click Event Listeners. These update the first and second values, set operators, and update the display. After the equal button is pressed, the calculate function is called to calculate the value.

The calculate function looks like this:

const calculate = (n1, operator, n2) => {
   const firstNum = parseFloat(n1)
   const secondNum = parseFloat(n2)
   if (operator === 'add') return firstNum + secondNum
   if (operator === 'subtract') return firstNum - secondNum
   if (operator === 'multiply') return firstNum * secondNum
   if (operator === 'divide') return firstNum / secondNum
 }

While relatively simple, this was a very educational project. It enforced both my CSS and JavaScript comprehension, and now I'll never need to Google math ever again!
