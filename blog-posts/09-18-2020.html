Blog Post - September 18th, 2020

This week, I watched a YouTube video from the channel Real Website Hints titled "5 Website Design Hints. Web Design Tutorial For Beginners." with tips on how to create professional-looking websites as an amateur developer.

The channel's tips were as follows:

  1. Create a color palette.
  2. Use great looking images.
  3. Add extra space around elements.
  4. Be decisive.
  5. Remember your mobile viewers.

While much of the content of Web Development focuses on how to use particular elements and attributes, I enjoyed the artistic element that the video introduced. Not only do web pages need to be functional and accessible; they should also be aesthetically pleasing with logical arrangement. While a web page developed with basic html elements may be physically readable, appropriate web design addresses readability and user engagement on a psychological level.

Perhaps two colors may be different enough that a user can distinguish them, but the developer should consider the psychological relationship between the colors. If two colors are similar shades, the user will mentally associate them. If two colors are on opposite positions on the color wheel, the user will mentally separate them.

Consider the following code: 

<p style="color: blue">Rey is friends with:</p>
<ul>
  <li style="color: blue">Finn</li>
  <li style="color: blue">Poe</li>
  <li style="color: red">Ben</li>
</ul>
<p style="color: red">Snoke is friends with:</p>
<ul>
  <li style="color: red">Phasma</li>
  <li style="color: red">Palpatine</li>
  <li style="color: blue">Hux</li>
</ul>

Which produces:

Rey is friends with:

  • Finn
  • Poe
  • Ben

Snoke is friends with:

  • Phasma
  • Palpatine
  • Hux

The lists explictly declare that Rey is friends with Finn, Poe, and Ben and that Snoke is friends with Phasma, Palpatine, and Hux; however, the colors suggest otherwise. Furthermore, the cultural context of the colors red and blue suggest that Rey and her friends are good while Snoke and his friends are not. The color palette has significant implications with psychological and cultural influences that the developer must keep in mind when designing a web page.

Real Website Hints also provides a resource for beginner programmers who lack the aesthetic intuition to create color palette: Paletton. Here, the user can select monochromatic, adjacent, triad, and tetrad color palettes. The website also provides a vision stimulation option where developers can preview how their palettes would appear on desaturated devices and to users with various forms of color blindness. This is an excellent resource to reference when creating accessible web pages!
